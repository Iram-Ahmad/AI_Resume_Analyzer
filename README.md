# AI Resume Analyzer

An AI-powered ATS Resume Screening System built with Python and Streamlit.

## Features

- Resume PDF Upload
- Skill Detection
- Field Detection
- ATS Match Score
- Job Description Matching
- AI Feedback
- PDF Report Download
- Dashboard Charts

## Technologies Used

- Python
- Streamlit
- PDFPlumber
- Matplotlib
- ReportLab

## Run Locally

pip install -r requirements.txt

streamlit run app.py
